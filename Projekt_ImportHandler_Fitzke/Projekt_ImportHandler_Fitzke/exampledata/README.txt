In diesem Ordner sind Beispieldaten.

In der importhandler.properties datei kann ein Ordner angegeben werden, 
aus dem die Daten importiert werden. Die Datei wird angelegt wenn das Program gestartet wird.

Standart ist c:\tmp\importdata.
Um die Daten aus diesem Ordner nutzen zu können bitte einen Ordner anlegen (c:\tmp\importdata) oder
und die Daten die man testen möchte da rein kopieren.

Aufruf des Programmes:
java -jar App help (dort gibts eine Übersicht)

java -jar App import (für Import)

Kommandos sind kombinierbar. Also
java -ja App clear outputdir import generate
